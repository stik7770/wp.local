== Description ==

This plugin creates the additional sections on the front page in themereviewer themes. A easy plugin to single page and post in add shortcode data for themes by Themereviewer Themes.  enjoy the full functionality of the Page Section For Themereviewer Software by installing the Page Section For Themereviewer.
Page Section For Themereviewer is a plugin created to enhance the functionality of the WordPress theme created by themerevieweringsoftware.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Page Section For Themereviewer
3. Activate Page Section For Themereviewer plugin from your Plugins page.
